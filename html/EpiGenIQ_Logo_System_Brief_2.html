04 — Logo Usage Rules

Do & Never Do

✓ Always Do
  • Use only the approved SVG source files — never recreate the wordmark
  • Maintain minimum clear space equal to the height of the icon mark on all sides
  • Use the primary dark variant on all dark backgrounds by default
  • Use the reverse light variant on white, bone, or light grey backgrounds
  • Apply the charge accent variant for campaign CTAs and high-energy moments
  • Apply the violet accent variant for AI outputs, scores, and intelligence content
  • Use the transparent versions when placing on photography with a color overlay
  • Include "™" symbol on first mention in every document, post, and page
  • Scale proportionally — never break the horizontal lockup proportions
  • Minimum size: 120px wide digital / 1.25 inches wide print
✕ Never Do
  • Never stretch, skew, rotate, or distort the logo in any direction
  • Never apply drop shadows, glows, or gradients to the logo
  • Never use unapproved color combinations — only the 8 approved variants
  • Never place on a busy photographic background without a solid color overlay
  • Never retype or recreate the wordmark in any other font
  • Never crowd the logo — always maintain clear space
  • Never reproduce smaller than 120px wide (digital) / 1.25" wide (print)
  • Never use the violet-only variant as a standalone brand color without approval
  • Never use the HealthStartz Plum Vitality colors (#3A1A4A, #7B3FA0) in EpiGenIQ materials
  • Never combine EpiGenIQ and HealthStartz logos without explicit co-brand authorization
